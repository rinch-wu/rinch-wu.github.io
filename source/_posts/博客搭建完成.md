## 博客搭建完成

这是我的第一篇博客文章！🎉

## 技术栈

- **静态博客框架**：[Hexo](https://hexo.io/)
- **主题**：[Fluid](https://hexo.fluid-dev.com/)
- **部署**：GitHub Pages
- **仓库结构**：
  - `source` 分支 → 保存 Hexo 源码
  - `main` 分支 → 存放生成的静态文件

## 搭建过程

从 0 开始搭建一个个人博客，整个过程其实并不复杂：

1. 安装 Hexo 和依赖
2. 配置主题和站点信息
3. 创建必要的页面（关于、标签、分类）
4. 配置 GitHub Pages 部署
5. 一键发布
